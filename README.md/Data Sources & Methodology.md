| Dataset            | Source                                                                              |
| ------------------ | ----------------------------------------------------------------------------------- |
| Airports           | Public aviation reference data                                                      |
| Countries          | ISO reference data                                                                  |
| Aircraft           | Public manufacturer specifications                                                  |
| Weather Categories | Based on aviation weather classifications                                           |
| Customers          | Generated using documented business rules                                           |
| Shipments          | Business-rule-generated because real shipment records are commercially confidential |
| Revenue            | Derived from shipment weight, route characteristics, and cargo category             |


Operational shipment data from commercial cargo airlines is not publicly available due to confidentiality. To simulate a realistic enterprise environment, operational datasets were generated using documented business rules while reference datasets were based on publicly available aviation information.
